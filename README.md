# StripeGolf

Интерактивный макет сервиса «Stripe Golf». Открывайте `index.html` или `golf_web.html`/`golf_mobile.html` — каждый файл самодостаточный и содержит весь интерфейс.

Верстка адаптивная: на мобильных устройствах используется нижняя навигация, на десктопе появляется вертикальное меню слева, а контент растягивается на всю доступную ширину. Никаких рамок телефона не требуется.

Разделы: «Главная» (профиль с новостным колокольчиком и чатом, настроек и поддержки), «Играть», «Карта», «Клубы» и «Турниры». Карта — Яндекс с переходом в слоты по клику на маркер. В «Клубах» у каждой карточки есть кнопка «Карта поля» с иллюстрацией конкретного поля (Skolkovo, Moscow CC, Pestovo, Agalarov, Tseleevo, Pirogovo с пометкой «лунка на воде» и «9 лунок»).

Слоты автоматически фильтруются при выборе дат, времени (утро 07:00–11:59, день 12:00–17:59, вечер 18:00–22:00), клуба, количества игроков и тегов (ти-тайм, тренировка, гольф-симулятор), а кнопки «Бронировать» фиксируют выбранные варианты прямо в макете. Набор слотов расширен, чтобы покрыть разные клубы и периоды дня, а в «Турнирах» можно зарегистрироваться одним нажатием.
