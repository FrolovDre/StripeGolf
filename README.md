# StripeGolf

Адаптивный макет сервиса «Stripe Golf» в классическом стиле приватного гольф-клуба. Открывайте `index.html` — это основной источник интерфейса. Файлы `golf_web.html` и `golf_mobile.html` просто перенаправляют на него для совместимости.

## Что внутри
- **Единый HTML** (`index.html`) подключает `styles.css` и `app.js`: стили вынесены в отдельный файл, а вся логика навигации, фильтров и данных — в JavaScript.
- **Адаптивная навигация**: снизу на мобильных устройствах и вертикальное меню слева на десктопе (с 768px). Активный раздел подсвечивается, показывается только его контент.
- **Разделы**: «Главная», «Играть», «Карта», «Клубы», «Турниры». В «Профиле» — колокольчик новостей, сообщения, настройки и поддержка; в остальных разделах — чистый контент.
- **Карта**: встроенный виджет Яндекс.Карт с реальными координатами клубов (Сколково, Пирогово, ForRest, Завидово, Агаларов) и быстрым переходом к слотам.
- **Клубы**: карточки с уникальными особенностями (например, «лунка на воде» и «9 лунок» у Пирогово) и кнопкой «Карта поля», которая открывает схематичную карту конкретного поля.
- **Играть**: слоты с автофильтрацией по дате (сегодня/завтра/выходные), времени (утро 07:00–11:59; день 12:00–17:59; вечер 18:00–22:00), клубу, количеству игроков и тегам («Ти-тайм», «Тренировка», «Гольф-симулятор»). Теги выделены разными цветами, фильтры применяются сразу при выборе.
- **Турниры**: карточки с кнопкой «Зарегистрироваться», меняющей состояние на «Вы зарегистрированы».

## Запуск
Откройте `index.html` в браузере. Подключения к сборщикам не требуется — всё работает на чистом HTML, CSS и JS.
