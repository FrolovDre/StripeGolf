# StripeGolf

Интерактивный мобильный макет сервиса «Stripe Golf». Открывайте `index.html` (или `golf_mobile.html`), чтобы увидеть рамку телефона; внутри неё отображается интерфейс из `golf_web.html`. Если GitHub Pages или просмотр через RAW недоступны, в оболочке автоматически подгружается встроенный макет.

Все три файла самодостаточны: основной интерфейс упакован внутрь iFrame и вшит в качестве резервной копии, поэтому отображение не зависит от внешних ресурсов и не будет «ломаться» при отсутствии доступа к `raw`/`pages`.

Внутри макета есть нижняя навигация по разделам: «Главная» (профиль), «Играть», «Карта», «Клубы» и «Турниры». Карта — отдельный экран с Яндекс.Картами, а в разделе «Клубы» у каждой карточки есть кнопка «Карта поля» с фокусом на схеме конкретного клуба (Skolkovo, Moscow CC, Pestovo, Agalarov, Tseleevo, Pirogovo с пометкой «лунка на воде» и «9 лунок»). Новости собраны в колокольчике справа сверху, а чат — слева сверху (оба доступны на домашней странице профиля, где остаются значки настроек и поддержки). Слоты автоматически фильтруются при выборе дат, времени (утро 07:00–11:59, день 12:00–17:59, вечер 18:00–22:00), клуба, количества игроков и тегов (ти-тайм, тренировка, гольф-симулятор), а кнопки «Бронировать» фиксируют выбранные варианты прямо в макете. Набор слотов расширен, чтобы покрыть разные клубы и периоды дня, а в «Турнирах» можно зарегистрироваться одним нажатием.
