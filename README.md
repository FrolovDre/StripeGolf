# StripeGolf

Интерактивный мобильный макет сервиса «Stripe Golf». Открывайте `index.html` (или `golf_mobile.html`), чтобы увидеть рамку телефона; внутри неё отображается интерфейс из `golf_web.html`. Если GitHub Pages или просмотр через RAW недоступны, в оболочке автоматически подгружается встроенный макет.

Все три файла самодостаточны: основной интерфейс упакован внутрь iFrame и вшит в качестве резервной копии, поэтому отображение не зависит от внешних ресурсов и не будет «ломаться» при отсутствии доступа к `raw`/`pages`.

Внутри макета есть нижняя навигация по разделам: «Карта», «Клубы», «Слоты» и «Новости». Карта и список клубов объединены в одном экране; используются реальные координаты Яндекс.Карт для отметок Skolkovo, Moscow CC, Pestovo, Agalarov, Tseleevo и Pirogovo. Слоты автоматически фильтруются при выборе дат, времени (утро 07:00–11:59, день 12:00–17:59, вечер 18:00–22:00), клуба, количества игроков и тегов (ти-тайм, тренировка, турнир), а кнопки «Бронировать» фиксируют выбранные варианты прямо в макете.
